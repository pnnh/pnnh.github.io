wt c++ web framework

https://www.webtoolkit.eu/wt/

```bash
cd ~/Library
git clone https://github.com/emweb/wt.git
git checkout 4.10.4
mkdir cmake-build && cd cmake-build
cmake -DCMAKE_INSTALL_PREFIX=/opt ..
make
sudo make install

```
