### sdkman命令列表

```bash
列出可以安装的Java列表
sdk list java 

安装指定版本的java
sdk install java 23-graalce

卸载指定版本的java
sdk uninstall java 23-graalce

设置为默认版本
sdk default java 23-graalce

在当前终端使用指定版本
sdk use java 23-graalce

列出本地安装的Java列表
sdk list java | grep installed
```
